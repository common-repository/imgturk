ImgTurk for WordPress is a lightweight and fast Instagram sidebar widget.

== Description ==

ImgTurk for WordPress lets you view any non-private tag or user profile from Instagram right on your WordPress website. It is designed to be clutter free and really easy to use; you do not even need to set up anything in Instagram to get going!

= Features =
* Easy to set up
* Lean and mean
* Tags and user profiles

== Installation ==

1. Install this plugin through the WordPress plugin directory, or by uploading it to your web server through the interface. If you can not upload it through the interface, use the file upload system provided by your host. Upload the files to /wp-content/plugins/
2. Active the plugin through the WordPress **Plugins** page
3. Add the widget to your sidebar in the **Appearance > Widgets**
4. Choose wether you want a **tag** or **user profile** and enter the tag or username (*without # or @*)
5. Done!
